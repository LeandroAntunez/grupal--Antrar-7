# Documentacion Tecnica  

>La documentación Técnica consiste en toda la información que nos explica cómo funciona un sistema, como está diseñado y con qué fin. Un documento de este tipo suele contener las características técnicas y la forma de operar del sistema. ésta información sirve para darle a entender a las personas que vallan a trabajar con él y de esta forma se le pueda dar mantenimiento.
